# 図解の日本語の違和感を言語化する（手伝い用フレーム）

インフォグラフィック HTML の品質を上げるため、違和感を**ラベル付きで記録**し、プロンプトやレビューに反映するためのテンプレです。

## 優先コンテキスト（プロジェクトで合意した前提）

| 項目 | 内容 |
|------|------|
| 主な成果物 | インフォグラフィック HTML（`generate_meeting_infographic_html.py`） |
| 読者 | 社外・クライアント向けにそのまま見せてよいレベル |
| いちばん避けたいこと | AI っぽい定型句・抽象語の連打で「人が書いた感じ」がしないこと |

## 違和感の観点（チェックリスト）

具体例をメモするとき、次の **1〜6 のどれに近いか**を付けます。

1. **トーン** — 教科書調と社内メモ調の混在、上から目線になっていないか。
2. **翻訳調・AI 定型** — 主語がぼんやりした文末の連発、抽象語だけが積み上がっていないか。
3. **冗長・リズム** — 同じ接続詞の連発、一文が長すぎる、見出しと本文の重複。
4. **用語の言い換え** — 説明が長すぎる、カタカナ語の連続、社内で使わないビジネス用語。
5. **会議文脈とのズレ** — 文字起こしとトーンの距離、未決が「決まった」ように読めるか。
6. **見出しと本文** — 見出しと内容がずれていないか。

## 記録テンプレ（コピペ用）

各「気になる箇所」につき 1 ブロック。

```text
- どの成果物か:
- 原文（そのまま貼る）:
- 違和感のラベル: （上記 1〜6。複数可）
- 自分ならこう書く: （1 文で可）
- 優先度: （高 / 中 / 低）
```

## 記入例（ダミー・形式サンプル）

実在の会議とは無関係です。**書き方の見本**として使ってください。

### 例 A（ラベル 2・AI 定型）

- **どの成果物か**: インフォ HTML の「価値」段落
- **原文（そのまま貼る）**: これらの施策を統合的に推進することが、事業成長の観点から極めて重要な役割を果たす仕組みとなっています。
- **違和感のラベル**: 2
- **自分ならこう書く**: 今回の会議では、A と B を同時に進める方針で合意しました。
- **優先度**: 高

### 例 B（ラベル 1・文体の混在）

- **どの成果物か**: インフォ HTML 見出しと本文
- **原文（そのまま貼る）**: （見出し）全体像 / （本文）本フローは三段階である。まず企画部が要件を整理します。
- **違和感のラベル**: 1
- **自分ならこう書く**: 見出し・本文ともにです・ますにそろえる。
- **優先度**: 高

### 例 C（ラベル 2・抽象語の連打）

- **どの成果物か**: インフォ HTML の「洞察」リスト
- **原文（そのまま貼る）**: 本質的な最適化と相乗効果により、包括的な価値創出が期待される。
- **違和感のラベル**: 2
- **自分ならこう書く**: 文字起こしにある「〇月までに△を紹介する」といった具体に置き換える。
- **優先度**: 中

### 例 D（ラベル 3・冗長）

- **どの成果物か**: インフォ HTML の導入セクション
- **原文（そのまま貼る）**: また、さらに、その結果として、再度確認すると、やはり重要なのは…（1 文が 120 字超）
- **違和感のラベル**: 3
- **自分ならこう書く**: 接続詞を減らし、文を 2 つに分ける。
- **優先度**: 中

### 例 E（ラベル 5・事実のズレ）

- **どの成果物か**: インフォ HTML の「決定」ボックス
- **原文（そのまま貼る）**: 来四半期までに契約を締結する。
- **違和感のラベル**: 5
- **自分ならこう書く**: 会議では期日まで話し合い中なら「文字起こし上は未確定です」と書く。
- **優先度**: 高
